┌─────────────────────────────────────────────────────────────────────────────────┐
│ FLUJO DE FUNCIONAMIENTO                                                          │
└─────────────────────────────────────────────────────────────────────────────────┘

1. Usuario accede a "Reservar Espacios" en el Dashboard

2. Ve la nueva sección "Buscar y Filtrar" con 4 campos:
   • Campo de texto: Nombre del Espacio
   • Dropdown: Tipo de Espacio
   • Campo numérico: Capacidad Mínima
   • Campo numérico: Capacidad Máxima

3. AL ESCRIBIR/SELECCIONAR:
   └─ Se aplican filtros en tiempo real
   └─ Se actualiza el listado dinámicamente
   └─ Se muestra contador: "X espacios encontrados"

4. SI NO HAY RESULTADOS:
   └─ Se muestra mensaje: "No se encontraron espacios"
   └─ Opción de hacer clic en "Limpiar Filtros"

5. ESPACIOS MOSTRADOS:
   └─ Solo espacios ACTIVOS (activo = true)
   └─ Cumplen con TODOS los criterios de filtro
   └─ Organizados por categoría (Labs, Canchas, Salas)
   └─ Con botones "Ver detalle" y "Reservar"

┌─────────────────────────────────────────────────────────────────────────────────┐
│ EJEMPLOS DE USO                                                                  │
└─────────────────────────────────────────────────────────────────────────────────┘

CASO 1: Buscar por nombre
   Usuario escribe: "Laboratorio"
   Resultado: Muestra todos los espacios con "laboratorio" en el nombre

CASO 2: Filtrar por tipo
   Usuario selecciona: "Canchas"
   Resultado: Muestra solo espacios de tipo Cancha

CASO 3: Filtrar por capacidad
   Usuario ingresa: Min=20, Max=50
   Resultado: Muestra solo espacios con capacidad entre 20 y 50 personas

CASO 4: Filtros combinados
   Usuario ingresa: Nombre="Sala", Tipo="Salas", Min=30
   Resultado: Muestra solo salas con "Sala" en el nombre y capacidad >= 30

CASO 5: Sin resultados
   Usuario aplica: Nombre="XXXXXXX", Tipo="Laboratorios"
   Resultado: "No se encontraron espacios que coincidan con los filtros"
              Opción: "Limpiar Filtros"

┌─────────────────────────────────────────────────────────────────────────────────┐
│ API ENDPOINT                                                                     │
└─────────────────────────────────────────────────────────────────────────────────┘

Ruta: GET /api/espacios/buscar

Parámetros de Query (todos opcionales):
  • nombre=string           - Búsqueda en nombre (LIKE)
  • tipo=string             - Filtrar por tipo
  • capacidad_min=number    - Capacidad mínima
  • capacidad_max=number    - Capacidad máxima

Ejemplos:
  GET /api/espacios/buscar?nombre=Lab
  GET /api/espacios/buscar?tipo=Canchas&capacidad_min=20
  GET /api/espacios/buscar?nombre=Sala&tipo=Salas&capacidad_min=30
